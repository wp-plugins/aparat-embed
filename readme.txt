Display Aparat videos and channels in WordPress.

== Description ==

با استفاده از این افزونه میتوانید ویدئوهای آپارات را به راحتی در سایت خود بگنجانید. پس از نصب افزونه کافیست آدرس ویدئو را در متن نوشته خود بچسبانید تا ویدئو به نوشته شما اضافه شود.
این افزونه همچنین امکان گنجاندن کانال (مجموعه ای از ویدئوها) را دارد.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can embed any Aparat videos or channels in your site. See: http://codex.wordpress.org/Embeds
4. Enjoy!
